xkcd_downloader
===============

Python modules required: Pillow
```shell
sudo pip install Pillow
```
Example-usage:

Fetch comics 150 through 170:
```shell
python xkcd-image.py 150-170
```
Fetch the most recent strip:
```shell
python xkcd-image.py 0
```
Fetch comic 188:
```shell
python xkcd-image 188
```
